# Zapret Simple Launcher 🚀

Удобная и легкая графическая оболочка (GUI) для управления утилитой [Zapret](https://github.com/flowseal/zapret-discord-youtube) под Windows. Написана на Python (PyQt6).

Лаунчер избавляет от необходимости каждый раз лезть в папку с программой, перебирать `.bat` файлы и держать открытыми черные окна командной строки (cmd).

## ✨ Главные фишки

* **Запуск в один клик:** Огромная анимированная кнопка Start/Stop. Программа сама запоминает путь к папке и последний выбранный вами скрипт.
* **Stealth-режим:** Лаунчер полностью скрывает черные окна командной строки при ручном запуске. Ничего не висит на панели задач.
* **Встроенные логи:** Весь текстовый вывод (stdout), который раньше сыпался в консоль, теперь аккуратно транслируется в реальном времени на вкладку **Logs**.
* **Управление службами (Services):** Скрипты для установки/удаления фоновых служб Windows (`service.bat`) автоматически отсортированы и вынесены на отдельную вкладку, чтобы не путаться.
* **Роутинг:** Возможность визуально вести список исключений (домены/IP).

## 📥 Использование (Готовый .exe)

Если вы не хотите устанавливать Python и собирать проект из исходников, вы можете скачать готовый скомпилированный файл из раздела [Releases](https://github.com/BigBoss649/Zapret_Simple_Launcher/releases).

1. Скачайте оригинальный [Zapret](https://github.com/flowseal/zapret-discord-youtube/releases) и распакуйте его в любую папку.
2. Скачайте и запустите `Zapret_Simple_Launcher.exe`.
3. Укажите путь к распакованной папке Запрета, выберите нужный скрипт (например, `general.bat`) и нажмите **START**.

> **⚠️ Важно:** Лаунчер автоматически запрашивает права Администратора при запуске (появится окно UAC). Это техническая необходимость — без админ-прав `zapret` не сможет загрузить сетевой драйвер `WinDivert`, а лаунчер не сможет скрыть окна консоли.

### 🛡️ Про антивирусы (False Positives)
Поскольку программа написана на Python, скомпилирована в один `.exe` файл (через PyInstaller) и требует права Администратора, некоторые параноидальные антивирусы могут выдавать **ложные срабатывания**. Весь исходный код полностью открыт в этом репозитории — вы можете изучить его и собрать `.exe` самостоятельно.

## 🛠️ Сборка из исходников

Если вы хотите запустить скрипт напрямую или собрать свой `.exe` файл, вам потребуется **Python 3.10+**.

**1. Клонирование репозитория:**
```bash
git clone [https://github.com/BigBoss649/Zapret_Simple_Launcher.git](https://github.com/BigBoss649/Zapret_Simple_Launcher.git)
cd Zapret_Simple_Launcher
```

**2. Установка зависимостей:**
```bash
pip install PyQt6 pyinstaller
```

**3. Запуск без компиляции:**
```bash
python zapret_manager.py
```

**4. Сборка в .exe файл:**
Выполните следующую команду в терминале, чтобы собрать независимый экзешник (он появится в папке `dist`):
```bash
python -m PyInstaller --noconsole --onefile --uac-admin --icon=logo.ico --add-data "logo.ico;." zapret_manager.py
```
